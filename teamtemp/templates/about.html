About Team Temperature

This application is designed to gather 'team temperature' - that is, our happiness score.

For your privacy, very little information is stored or recorded. Each submitter is represented by an anonymous random ID, and a cookie is stored only to protect against accidental double submissions from the same user with the same browser.

The results page is available to the owner using the same cookie mechanism as well as a password in case the cookie is removed.

It is deployed at teamtempapp.herokuapp.com hosted on Heroku and the source for this version can be examined on Github. All reasonable pull requests will be considered, as will any issues raised.

The application is written in Django against a PostgreSQL database.
